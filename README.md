#  Lista de Tareas

##  Descripción

Aplicación sencilla en Python con Tkinter para organizar tareas diarias.
Permite agregar, completar y eliminar tareas de forma rápida usando botones o teclado.

## Funciones

* Agregar tareas
* Completar tareas
* Eliminar tareas

##  Atajos

* Enter → Agregar
* C → Completar
* Delete → Eliminar
* Esc → Salir

##  Uso

```bash
python lista_tareas.py
```
